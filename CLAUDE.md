# AGV-TDPS 项目说明

## 项目概述
STM32F103C8 底盘控制 + Orange Pi 4 Pro 上位机，实现自主巡线、弯道控制、圆圈陷阱规避。

## 硬件信息

### STM32F103C8
- 开发工具：Keil5 + STM32CubeMX
- 主频：72MHz（HSE 8MHz × PLL 9倍）
- 烧录：ST-Link

### 电机驱动：TB6612
| 引脚 | STM32 | 功能 |
|------|-------|------|
| PWMA | PA0 (TIM2_CH1) | 左轮速度 |
| PWMB | PA1 (TIM2_CH2) | 右轮速度 |
| AIN1 | PA2 | 左轮方向 |
| AIN2 | PA3 | 左轮方向 |
| BIN1 | PA4 | 右轮方向 |
| BIN2 | PA5 | 右轮方向 |
| STBY | PA6 | 芯片使能 |

### PWM 配置（TIM2）
- Prescaler=71, Period=999 → 1kHz，占空比范围 0-999

### 传感器：8路红外 + CD4051 多路复用
| 引脚 | STM32 | 功能 |
|------|-------|------|
| AD0  | PB12  | 地址线 bit0 |
| AD1  | PB13  | 地址线 bit1 |
| AD2  | PB14  | 地址线 bit2 |
| OUT  | PB15  | 传感器输出 |

### Orange Pi 4 Pro
- SSH：`orangepi@172.24.12.122`
- 串口：`/dev/ttyS7`（引脚 6/8/10）
- Python：3.10.12，pyserial 3.5 已安装

## 通信协议

### 上位机 → STM32
格式：`$<CMD>[PARAM]#<XOR>\n`

| 指令 | 含义 |
|------|------|
| T | 切换自动巡线模式 |
| S | 停止，切回手动 |
| F | 手动前进 |
| B | 手动后退 |
| L | 原地左转 |
| R | 原地右转 |
| V\<speed\> | 设置基础速度（1-999），如 V500 |
| H | 心跳包（每300ms发一次） |

XOR 校验范围：`$` 到 `#` 之间所有字节（含两端）。

### STM32 → 上位机
格式：`$S<mode>,<lspd>,<rspd>,<sensor_hex>,<error>#<XOR>\r\n`

每 200ms 上报一次。

## 控制算法

### PD 巡线
- Kp=120, Kd=80，控制周期 20ms（50Hz）
- 误差范围：-3.5（最左）到 +3.5（最右），0 为正中

### 弯道自适应降速
| 误差绝对值 | 速度 |
|-----------|------|
| < 1.0 | base_speed（全速） |
| 1.0 ~ 2.5 | base_speed × 70% |
| > 2.5 | base_speed × 45% |

### 心跳超时
500ms 内未收到任何帧，自动停车并回报 `!TIMEOUT`。

## 文件结构
```
Core/Src/
  main.c      # 主控逻辑：帧解析、PD控制、状态上报
  motor.c     # TB6612 电机驱动
  sensor.c    # CD4051 传感器读取 + 加权平均误差
orange_pi/
  agv_controller.py  # 橙派上位机：帧收发、心跳、状态解析
```
